CRC - Unpacking procedure
CRC downloads are in TAR archive format and compressed with GZIP. To unpack a download in this format, follow these instructions.
  1. For this example we will assume you downloaded the archive CRC.linux.tar.gz.
  2. If required, create an appropriate sub-directory and place the archive there using the mkdir and mv unix commands.
  3. Use gunzip CRC.linux.tar.gz to decompress the zipped file.
  4. Unpack the CRC archive by running tar xvf CRC.linux.tar (without the .gz extension). This will extract the CRC package to the current directory.
